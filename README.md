進捗どうですか

## Installation

1. `npm install hubot-shinchoku --save`
2. "hubot-shinchoku" を external-scripts.json に追加
4. Reboot Hubot

## Commands

```
shinchoku - 進捗画像をランダムに返す
進捗 - 進捗画像をランダムに返す
```
